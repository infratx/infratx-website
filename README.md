\# InfraTX Systems - Static Website (Demo package)



This package is a multi-page static website for \*\*InfraTX Systems\*\*.

It includes:

\- index.html, about.html, services.html, cloud.html, security.html, contact.html

\- CSS in `/css/styles.css`

\- JS in `/js/main.js`

\- assets in `/assets` (replace the placeholder logo.png with your uploaded logo)



Notes:

\- The package is a static HTML/CSS/JS site. It can be hosted on GitHub Pages, Azure Static Web Apps, or any static host.

\- Replace `/assets/logo.png` with the final logo you uploaded earlier (keep filename `logo.png`).

\- The contact form is a demo. Replace the form `action` with your backend or a form handling service.

\- To preview locally, open `index.html` in a browser, or serve with a local static server:

&nbsp; `python -m http.server` (from the package root)
